# 最好用的串口调试工具

## 简介
本仓库提供了一款功能强大的串口调试工具，适用于串口程序开发过程中的调试工作。该工具支持自发自接、定时发送、Asc码和十六进制发送与接收，是串口调试的理想选择。

- **Asc码和十六进制发送与接收**：支持Asc码和十六进制两种数据格式的发送与接收，满足不同调试需求。
